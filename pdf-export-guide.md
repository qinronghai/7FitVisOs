# 7Fit PDF 导出参数建议（v3）

## 通用参数（推荐）
- 纸张：A4（210mm × 297mm）
- 方向：纵向（Portrait）
- 缩放：100%
- 页边距：上 14mm / 下 14mm / 左 12mm / 右 12mm
- 打印背景图形：开启

## 页眉页脚建议
- 页眉：关闭（避免默认 URL 干扰）
- 页脚：可选开启
  - 左：7Fit
  - 中：文档名 + 版本（如 UI Kit v3.0）
  - 右：Page X of Y

## 浏览器导出流程
1. 打开目标 html
2. Ctrl+P（或 Cmd+P）
3. 目标选择“另存为 PDF”
4. 在 More settings / 更多设置中按上面参数配置

## 当前页面兼容性
- 三个页面都内置 `@media print`
- 自动隐藏主题切换等交互控件
- 自动切换到打印友好高对比样式
